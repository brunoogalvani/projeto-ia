Projeto IA

A inteligencia artifical é software com um algoritmo capaz de realizar tarefas semelhante a um humano. Ela raciocina, aprende e resolve problemas de forma autonoma.
Além disso, ela possui uma grande quantidade de dados para poder realizar todas essas tarefas e que, também, é capaz de "ensinar" essas IAs de uma forma mais eficiente.

Outras funções das inteligências artificiais são: Ter voz própria, reconhecimento facial, logística e outras. Em algoritmos de Machine Learning supervisionados, o objetivo é criar um modelo que relacione entradas e saídas desejadas após o treinamento. Já nos algoritmos não supervisionados, não há uma saída desejada e o modelo identifica padrões nos dados. O aprendizado por reforço envolve algoritmos que aprendem a tomar decisões interagindo com o ambiente e recebendo feedback em forma de recompensas ou punições.
